# Color Palette Extractor

Upload an image and extract its dominant color palette automatically.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Extractor** (or Skip).

**Step 3:** Click the upload area to select an image from your device (JPG, PNG, GIF, WebP).

**Step 4:** The image is analyzed and the 8 most dominant colors are extracted.

**Step 5:** The color palette bar shows the colors visually. Hover for hex codes.

**Step 6:** Click any color card to copy its HEX code to clipboard.

**Step 7:** Click **Copy All Colors** to copy all hex codes at once.

## Features
- Image upload (JPG, PNG, GIF, WebP)
- Automatic color extraction (8 dominant colors)
- Visual palette bar with hover tooltips
- Color cards with HEX and RGB values
- Click to copy individual colors
- Copy all colors at once
- Canvas-based image processing

## Tech
HTML5 Canvas + FileReader API + Clipboard API
